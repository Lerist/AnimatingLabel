# AnimatingLabel

Adds animation feature for numeric values in UILabel. 

![](img/gif.gif)

## Usage

```swift
let label: UILabel

label.animate(to: 5.0)
label.animate(to: 5.0, duration: 1.0)
label.animate(to: 5.0, duration: 1.0, formatter: NumberFormatter(), easingOption: .easeInOut)
```
